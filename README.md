# EDA-Exploratory-Data-Analysis-
# 📊 Loan Approval Exploratory Data Analysis (EDA)

A comprehensive Exploratory Data Analysis (EDA) project built using Python to uncover patterns, trends, and relationships within a loan approval dataset. The analysis focuses on applicant demographics, income distribution, loan characteristics, and factors influencing loan approval decisions through statistical summaries and visualizations.
# Short Description / Purpose

The **Loan Approval EDA Project** is designed to analyze a loan application dataset and identify the key factors that influence loan approval outcomes. Using Python libraries for data analysis and visualization, the project helps uncover insights that support better lending decisions and data-driven financial analysis.
# Tech Stack
This project was developed using the following tools and technologies:
* 🐍 **Python** – Core programming language for data analysis.
* 📊 **Pandas** – Data loading, cleaning, manipulation, and analysis.
* 📈 **Matplotlib** – Data visualization and plotting.
* 🎨 **Seaborn** – Statistical data visualization and distribution analysis.
* 📓 **Jupyter Notebook** – Interactive environment for analysis and documentation.
* 📁 **Dataset Format** – CSV (`.csv`) for raw data and Notebook (`.ipynb`) for implementation.
# 📂 Data Source
**Source:** Loan Approval Dataset (CSV)
The dataset contains loan application records with applicant details such as:
* Applicant Income
* Co-applicant Income
* Loan Amount
* Loan Term
* Credit History
* Gender
* Marital Status
* Education
* Self-Employment Status
* Property Area
* Loan Approval Status
The data is analyzed to understand how these factors influence loan approval decisions.
